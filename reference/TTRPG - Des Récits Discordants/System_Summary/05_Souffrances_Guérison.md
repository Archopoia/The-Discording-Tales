# Souffrances et Guérison

## Les 8 Souffrances

Chaque Souffrance est liée à un Attribut :

1. **Blessures (FOR)** : Dégâts physiques
2. **Fatigues (AGI)** : Épuisement, lenteur
3. **Entraves (DEX)** : Empêchements, blocages
4. **Disettes (VIG)** : Faim, soif, manque
5. **Addictions (EMP)** : Dépendances, attachements malsains
6. **Maladies (PER)** : Affections, infections
7. **Folies (CRÉ)** : Troubles mentaux, obsessions
8. **Rancœurs (VOL)** : Ressentiments, haines

## Dés de Souffrance (DS)

### Calcul
- **1-2 DS** : Léger
- **3-5 DS** : Modéré → Séquelle Passagère
- **6-9 DS** : Sévère → Séquelle Durable
- **10-11 DS** : Critique → Séquelle Permanente
- **15-20 DS** : Extrême → Séquelle Fatale
- **21+ DS** : Vaincu (Coma/Folie)
- **26+ DS** : Mort ou Perdu

### Application
- DS appliqués comme **Dés Négatifs** aux jets liés à l'Attribut affecté
- Accumulation progressive

## Séquelles

### Types de Séquelles

#### Passagère (3-5 DS)
- Effets temporaires
- Guérit avec le temps

#### Durable (6-9 DS)
- Effets à long terme
- Peut nécessiter traitement spécialisé

#### Permanente (10-11 DS)
- Effets permanents
- Modifie les capacités du personnage

#### Fatale (15-20 DS)
- Menace de mort
- Nécessite intervention immédiate

### Effets des Séquelles
- Dés Négatifs supplémentaires
- Pénalités d'Attributs
- Limitations physiques/psychologiques
- **Pitié** : Gain d'influence avec Séquelles

## Résistance

### Compétences Résistantes
Chaque Souffrance a une Compétence Résistante :
- **Blessures** : [Robustesse]
- **Fatigues** : [Satiété]
- **Entraves** : [Rectitude]
- **Disettes** : [Satiété]
- **Addictions** : [Rectitude]
- **Maladies** : [Immunité]
- **Folies** : [Rectitude]
- **Rancœurs** : [Rectitude]

### Résistance passive (pas de jet)
- La résistance est **toujours passive** : le **Niv** de la Compétence Résistante (plus Niv d'équipement lié) **absorbe** autant de DS (p. 72 : « Vous absorberez passivement un nombre de Souffrances égal à votre Niv de CT de Résistance liée au mal »).
- Il n’y a **jamais** de « Jet de Résistance » : on n’effectue pas de jet pour résister ; on applique le Niv pour réduire les DS subis.

## Rage et Évanouissement

### Rage (10+ Souffrances totales)
- **Jet de Rage** : 1d6, doit être > Niv de Rage pour agir contre l'instinct
- **Échec** : Action instinctive, DS convertis en Dés Positifs pour cette action
- **6** : Peut "Guider" la Rage

### Évanouissement/Démence (15+ Souffrances)
- **Jet d'Évanouissement** : 1d6, doit être > Niv d'Évanouissement pour agir
- **Échec** : Inconscient ou fou

### Vaincu (21+ Souffrances)
- Coma ou Folie
- Situation de Défaite

### Mort ou Perdu (26+ Souffrances)
- Mort physique ou perte totale de l'identité

## Instinct

### Conversion des DS
- En situation critique de survie
- DS convertis temporairement en Dés Positifs
- Pour cette action uniquement

## Guérison

### Guérir sur le Temps

#### Jours de Souffrance
Chaque DS nécessite un certain nombre de "Jours de souffrance" à guérir :
- Basé sur le niveau de Souffrance
- Modifié par la résistance, l'environnement, le repos, la nourriture

#### Jet de Repos
- Quotidien
- Détermine combien de jours de souffrance sont guéris
- Modifié par :
  - Résistance
  - DS actuels
  - Environnement
  - Type de repos
  - Nourriture/boisson

### Traitements

#### Types de Traitements
- **Niv 1** : Remèdes simples (désinfectant, repos)
- **Niv 2** : Traitements de base (bandages, potions simples)
- **Niv 3** : Soins spécialisés (chirurgie mineure, médecine)
- **Niv 4** : Soins avancés (chirurgie majeure, traitements complexes)
- **Niv 5** : Soins exceptionnels (chirurgie générale, médecine sur mesure)

#### Traitements par Souffrance
Chaque Souffrance a des traitements spécifiques listés dans les tableaux.

#### Surdosage
- Trop de traitements → nouvelles Addictions ou Maladies

### Restaurations (Nourriture/Boisson)

#### Aliments
- **Viandes** : Restaurent Blessures
- **Noix** : Restaurent Entraves
- **Céréales** : Restaurent Fatigues
- **Fruits** : Restaurent Disettes
- **Légumes** : Restaurent Maladies
- **Épices** : Restaurent Addictions
- **Miel** : Restaure Folies
- **Alcool** : Restaure Rancœurs

#### Préparation
- **Haché/Émincé** : Modifie les effets
- **Grillé/Rôti** : Ajoute des effets
- **Cuit** : Améliore la restauration

### Traitements par Motivation/Personnalité

#### Motivations (Gens)
- Actions sur une Pentaine (5 jours)
- 3 niveaux (-), (0), (+)
- Traitent les Souffrances liées

#### Motivations (Choses)
- Similaire mais orienté vers les objets/activités

#### Personnalités
- Actions basées sur les traits de personnalité
- Traitent les Souffrances via le rôle-play

### Soutiens
- **Soigneur** : Personnage spécialisé en soins
- Optimise la guérison
- Peut fournir des traitements

## Situation de Défaite

### Table de Défaite
L'Éveilleur tire sur une table pour déterminer :
- Où le personnage se réveille
- Dans quel état
- Ce qui lui manque

### Perdre ses Affaires
- Objets de valeur volés ou perdus
- Niv d'Abondance augmente avec le temps
- Plus difficile à retrouver
