# Exemples de Personnages - Des Récits Discordants

## Personnage 1 : Slaadéen Felichrem - Artiste Audacieux

### Concept
Un artiste nomade qui cherche à capturer la beauté de la souffrance dans ses œuvres.

### Création Rapide

**Peuple** : Slaadéen
**Race** : Felichrem
**Sexe** : Autre
**Âge** : Jeune (20 ans)

**Attributs** (Base + Variation) :
- FOR: +1
- AGI: +2
- DEX: +1
- VIG: 0
- EMP: +3 (plus élevé)
- PER: +1
- CRÉ: +2
- VOL: 0

**Symbole** : Charisme (EMP le plus élevé)
- **Symbolique** : Charisme (3 Actions, 3 Marques Éternelles)
- **Fascinante** : Réflexion (2 Actions, 2 Marques Éternelles)
- **Intéressante** : Aisance (1 Action, 1 Marque Éternelle)

**Compétences Éduquées** :
- Populaire : [Inspiration] +1D
- Populaire : [Débrouillardise] +1D
- Populaire : [Chant] +1D
- Raciale : [Rectitude] +2D
- Raciale : [Apprivoisement] +2D
- Familiale : [Puissance] +3D

**Compétences Exprimées** (10 Dés) :
- [Inspiration] : +3D (Symbolique)
- [Narration] : +2D (Symbolique)
- [Séduction] : +1D (Symbolique)
- [Artisanat] : +2D (Fascinante)
- [Nature] : +1D (Fascinante)
- [Danse] : +1D (Intéressante)

**Valeurs** :
- Moralité : Thrasonirie (Audace Rêveuse) +6D
- Motivation : Générosité (envers Gens) +3D
- Personnalité : Extraverti (Charisme) +1D

**Traits** :
- Socialité : Autre (Unique) +6D
- Disposition : Slaadéens +3D
- Passion : Art (création artistique) +1D

**Titre** :
- Niv de Gouvernance : +1 (Chefferie Nomade)
- Dés d'Influence : +2
- Relation : Un mécène Aristois (+2 Influence)

**Langues** :
- Tiuorr : Parlé Exercé, Écrit Essentiel, Signé Enfantin

**Richesses** :
- 1000 cO (équipement artistique, matériaux, provisions)

### Histoire
Né dans une caravane nomade Slaadéenne, ce personnage a toujours été 
fasciné par la beauté qui émerge de la souffrance. Leur art capture 
les moments où douleur et plaisir se rencontrent, où les Souffrances 
deviennent sources d'inspiration.

### Style de Jeu
- Utilise [Inspiration] pour motiver les autres
- Utilise [Narration] pour raconter des histoires
- Utilise [Artisanat] pour créer des œuvres
- Les Souffrances deviennent sources d'inspiration artistique
- Les Valeurs guident la création artistique

## Personnage 2 : Aristois Monte-Foudre - Guerrier Souverain

### Concept
Un guerrier qui cherche à prouver sa valeur par la force, mais qui apprend que la vraie force vient de la maîtrise de ses Souffrances.

### Création Rapide

**Peuple** : Aristois
**Race** : Monte-Foudre
**Sexe** : Masculin
**Âge** : Adulte (30 ans)

**Attributs** :
- FOR: +3 (plus élevé)
- AGI: +1
- DEX: 0
- VIG: +2
- EMP: +1
- PER: 0
- CRÉ: -1
- VOL: +2

**Symbole** : Puissance (FOR le plus élevé)
- **Symbolique** : Puissance (3 Actions, 3 Marques Éternelles)
- **Fascinante** : Athlétisme (2 Actions, 2 Marques Éternelles)
- **Intéressante** : Domination (1 Action, 1 Marque Éternelle)

**Compétences Éduquées** :
- Populaire : [Puissance] +1D
- Populaire : [Chevauchement] +1D
- Populaire : [Lutte] +1D
- Raciale : [Armé] +2D
- Raciale : [Chevauchement] +2D
- Familiale : [Armé] +3D

**Compétences Exprimées** (10 Dés) :
- [Armé] : +3D (Symbolique)
- [Désarmé] : +2D (Symbolique)
- [Improvisé] : +1D (Symbolique)
- [Port] : +2D (Fascinante)
- [Grimpe] : +1D (Fascinante)
- [Commandement] : +1D (Intéressante)

**Valeurs** :
- Moralité : Trèpharchie (Croissance Souveraine) +6D
- Motivation : Réussite (envers Gens) +3D
- Personnalité : Véhément (Puissance) +1D

**Traits** :
- Socialité : Masculinité (Honneur) +6D
- Disposition : Aristois +3D
- Passion : Combat (se battre) +1D

**Titre** :
- Niv de Gouvernance : +2 (Tribu)
- Dés d'Influence : +3
- Relations : 2 alliés (+3 Influence chacun)

**Langues** :
- Eskandois : Parlé Exercé, Écrit Essentiel, Signé Enfantin

**Richesses** :
- 1000 cO (armes, armure, monture, provisions)

### Histoire
Élevé dans une tribu guerrière, ce personnage a toujours cherché à 
prouver sa valeur par la force. Mais après avoir subi de nombreuses 
Blessures et être entré en Rage plusieurs fois, il apprend que la 
vraie force vient de la maîtrise de ses Souffrances, pas de leur 
évitement.

### Style de Jeu
- Utilise [Armé] pour combattre
- Utilise [Commandement] pour diriger
- Les Blessures créent des situations de défi
- La Rage doit être gérée, pas évitée
- Les Valeurs guident les décisions de combat

- DEX: +1
- VIG: 0
- EMP: +1
- PER: +2
- CRÉ: +3 (plus élevé)
- VOL: +2

**Symbole** : Réflexion (CRÉ le plus élevé)

**Valeurs** :
- Moralité : Mignusophie (Sagesse Imbriquée) +6D
- Motivation : Fondation (envers Choses) +3D

- Disposition : Griscribes +3D
- Passion : Connaissance (apprendre) +1D

**Titre** :
- Niv de Gouvernance : +3 (Chefferie)
- Dés d'Influence : +4
- Relations : 3 alliés (+4 Influence chacun)

**Langues** :
- Kuhkcherd : Parlé Exercé, Écrit Essentiel, Signé Enfantin
- Eskandois : Parlé Essentiel, Écrit Essentiel

**Richesses** :
- 1000 cO (livres, outils, provisions)

Étudiante perpétuelle, cette personnage cherche à comprendre les 
interconnexions entre toutes choses. Chaque Souffrance révèle une 
vérité sur le monde. Chaque échec enseigne quelque chose. La sagesse 
vient de l'acceptation de la complexité.

### Style de Jeu
- Utilise [Investigation] pour découvrir
- Utilise [Nature] pour comprendre le monde
- Utilise [Médecine] pour soigner
- Les Souffrances révèlent des vérités
- Les Valeurs guident la recherche de connaissance

## Créer Votre Propre Personnage

1. **Choisissez un concept** : Qui est votre personnage ?
2. **Choisissez Peuple/Race** : Qui correspond au concept ?
3. **Déterminez Attributs** : Quels sont ses points forts/faibles ?
4. **Choisissez Compétences** : Que sait-il faire ?
5. **Déterminez Valeurs/Traits** : Qu'est-ce qui le motive ?
6. **Créez l'histoire** : Comment en est-il arrivé là ?

**N'oubliez pas** : Avec la méthode Amnésique, vous pouvez créer 
progressivement. Commencez simple, découvrez le reste en jouant !
