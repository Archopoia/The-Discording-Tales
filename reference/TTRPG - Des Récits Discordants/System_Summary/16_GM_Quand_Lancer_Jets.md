# Quand lancer quels jets — Guide MJ

*Sources : 01_Systeme_General, 03_Attributs_Aptitudes_Competences, 05_Souffrances_Guérison, 06_Combat, 08_Temps_Labeurs, 09_Groupe_Ambiance, 10_Voyages et sections MJ du livre.*

Ce document indique **quand** demander quel jet (ou appliquer une règle sans jet) pour chaque type de situation. Utilisez toujours le format exact : **Roll [Compétence] vs Niv +X**. La Compétence doit être l’une des 72 (ex. [Grimpe], [Négociation]) ; jamais un Attribut ou une Aptitude seule dans les crochets.

## Action à issue incertaine

- **Quand** : Le personnage tente une action avec chance d'échec.
- **Jet** : Roll [Compétence] vs Niv +X — la Compétence doit être l'une des 72 (ex. [Grimpe], [Négociation], [Investigation]). Jamais un Attribut ou une Aptitude dans les crochets.
- **Réf.** : 01, 03.

## Jet Précisé (Niveau de Temps)

- **Quand** : Le personnage prend du temps pour une action ; vous pouvez offrir de réduire la difficulté en échange de plus de NdT.
- **Règle** : **"Faire 0"** = pas de jet, résultat basé sur les niveaux. Sinon : +1 NdT = -1 Niv d'Épreuve (ex. Roll [Artisanat] vs Niv +0 si le joueur accepte +1 NdT).
- **Réf.** : 01, 08.

## Labeurs (journée de travail, artisanat, entraînement, récupération)

- **Quand** : Une journée de Labeur (8 h de travail, artisanat, entraînement, recherche, récupération).
- **Jet** : Roll [Compétence] appropriée vs Niv +X — ex. Roll [Artisanat] vs Niv +1 pour une journée de craft, Roll [Médecine] vs Niv +0 pour soins, Roll [Géographie] pour recherche.
- **Réf.** : 08.

## Surmenage (plus d'un Labeur par jour)

- **Quand** : Le personnage effectue plus d'un Labeur par jour.
- **Jet** : **Aucun**. La résistance est passive : le Niv de la Compétence Résistante (ex. [Satiété] pour Fatigues) absorbe les DS ; pas de jet de résistance.
- **Réf.** : 08, 05.

## Voyage — Navigation / perte de chemin

- **Quand** : Navigation, orientation, risque de se perdre en voyage.
- **Jet** : Roll [Géographie] ou Roll [Estimation] ou Roll [Vision] vs Niv +X (ex. Roll [Géographie] vs Niv +2). Échec = perdu, temps perdu, rencontres possibles.
- **Réf.** : 10_Voyages.

## Voyage — Traverser un milieu avec Conditions

- **Quand** : Traverser un milieu hostile (Conditions : froid, chaleur, obscurité, etc.). Voir milieux p.177 du livre.
- **Règle** : Les Niv d'Outil "absorbent" les Niv de Condition. S'il reste des Niv de Conditions non absorbés : jet avec Dés d'Outil ET d'Habituation vs Épreuve = Niv de Temps voyagé + Niv de Conditions. Résistance aux DS passive (Niv) — pas de jet de résistance.
- **Réf.** : Livre p.102 ; 10_Voyages.

## Rencontres (voyage)

- **Quand** : Déterminer une rencontre pendant le voyage.
- **Règle** : Fréquence selon temps de voyage et terrain ; l'Éveilleur peut utiliser une table ou un jet selon le contexte. Adapter au lieu et au récit.
- **Réf.** : 10_Voyages, 14_Monde_Iaodunei.

## Groupe — Cohésion (Jet d'Ambiance)

- **Quand** : Situations significatives pour la cohésion du groupe (séparations, conflits internes, succès/échecs collectifs).
- **Jet** : Roll [Ambiance] vs Niv +X. Mécanique : 5dD (dés discordants), résultat = somme des 5, succès si somme ≥ Niv. Toujours inclure la ligne parseable pour le bouton Roll.
- **Réf.** : 09_Groupe_Ambiance.

## Groupe — Action collective (Jet de Groupe)

- **Quand** : Action collective ; chaque membre fait un jet, modifié par l'Ambiance du groupe ; résultats moyennés.
- **Jet** : Chaque joueur : Roll [Compétence] vs Niv +X ; modifier par le pool d'Ambiance ; succès si moyenne ≥ Niv.
- **Réf.** : 09.

## Groupe — Un seul tente l'action (Jet d'Audace)

- **Quand** : Un individu tente une action de groupe seul.
- **Jet** : Roll [Compétence] vs Niv +X avec pénalité : -1 Dé par membre du groupe au-delà de 1.
- **Réf.** : 09.

## Combat — Initiative / qui proagit

- **Quand** : Déterminer qui peut agir de manière proactive (Clins d'Alerte).
- **Jet** : Roll [Fluidité] (ou compétence appropriée) vs Niv.
- **Réf.** : 06_Combat.

## Combat — Attaque

- **Quand** : Frapper, tirer, neutraliser.
- **Jet** : Roll [Armé], [Désarmé], [Improvisé], [Lutte], [Bottes], [Ruses], [Bandé], [Propulsé], [Jeté] selon l'action — vs Niv +X.
- **Réf.** : 06, 03.

## Combat — Jet de Souffle (tous les -10 Clins)

- **Quand** : À chaque -10 Clins en combat, tous font un Jet de Souffle (Domination [Endurer]).
- **Jet** : Roll [Gloutonnerie] ou Roll [Beuverie] ou Roll [Entrailles] vs Niv +X — utiliser une **Compétence** (pas l'Aptitude [Endurer]). Le joueur choisit laquelle selon le contexte.
- **Réf.** : Livre p.086 ; 06_Combat.

## Combat — Réactions (Bloquer, Parer, Esquiver)

- **Quand** : Défense contre une attaque.
- **Jet** : Roll [Armé] (Parade) ou compétence de réaction selon les règles combat ; coût en Niv d'Ébranlement.
- **Réf.** : 06.

## Infiltration / Discrétion (stealth)

- **Quand** : Passer inaperçu, voler sans être vu, se faufiler, filature, embuscade.
- **Jet** : Roll [Dissimulation], Roll [Escamotage], ou Roll [Illusions] vs Niv +X selon l'action.
- **Réf.** : 03 (Aisance → Infiltration).

## Social (négociation, séduction, intimidation, commandement)

- **Quand** : Négocier, séduire, intimider, commander, convaincre, débattre.
- **Jet** : Roll [Négociation], [Séduction], [Intimidation], [Commandement], [Obéissance], [Obstinance], [Présentation], [Chant], [Inspiration], etc. vs Niv +X — selon l'approche.
- **Réf.** : 03 (Charisme, Domination).

## Artisanat / Cuisine

- **Quand** : Création d'objets, préparation de repas, artisanat alimentaire.
- **Jet** : Roll [Artisanat] vs Niv +X (Artisanat inclut Alimentaire, des Graisses, du Papier, des Plantes, du Textile, du Cuir, etc.).
- **Réf.** : 03, 11_Possessions_Equipement.

## Marché / trouver acheteur ou vendeur

- **Quand** : Trouver un marché ou un partenaire commercial ; négocier un prix.
- **Jet** : Roll [Géographie] (+/- NdT de recherche) pour trouver un marché ; Roll [Négociation] pour le prix / courtisanat.
- **Réf.** : 10, 11 ; livre p.098.

## Rage (10+ Souffrances totales)

- **Quand** : Le total des 8 Souffrances atteint 10+ ; le personnage doit tester pour agir contre l'instinct.
- **Jet** : Roll [Rage] vs Niv +X (X = Niv de Rage). Mécanique : 1d6 > Niv (succès si résultat > Niv). Inclure la ligne parseable.
- **Réf.** : 05_Souffrances_Guérison.

## Évanouissement (15+ Souffrances)

- **Quand** : Le total des Souffrances atteint 15+ ; le personnage doit tester pour agir (sinon inconscient ou fou).
- **Jet** : Roll [Évanouissement] vs Niv +X. Mécanique : 1d6 > Niv. Inclure la ligne parseable.
- **Réf.** : 05.

## Guérison quotidienne (Jet de Repos)

- **Quand** : Fin de journée / repos ; déterminer combien de jours de souffrance sont guéris.
- **Jet** : Roll [Repos] vs Niv +X. Mécanique : 5dD, somme vs Niv. Inclure la ligne parseable.
- **Réf.** : 05.

## Résistance aux Souffrances (DS)

- **Quand** : Les règles disent que la résistance réduit les DS subis.
- **Jet** : **Aucun**. La résistance est **toujours passive** : le Niv de la Compétence Résistante (Robustesse, Satiété, Rectitude, Immunité) plus Niv d'équipement absorbe les DS. Ne jamais demander de "Jet de Résistance".
- **Réf.** : 05 ; livre p.72.
