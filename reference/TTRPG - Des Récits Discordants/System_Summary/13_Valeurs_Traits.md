## Valeurs

### Concept
- **Valeurs** : Besoins qui interprètent le monde
- Guérissent ou définissent les **Folies**
- Chaque Valeur a des Dés associés
- Se marquent à chaque Échec (comme les Compétences)

### Types de Valeurs

#### Moralité (+6D)
- **Héritée du Peuple** : Transgénérationnelle
- Inconsciente et transmise
- Détermine ce qui est choquant
- **Exemples** :
  - **Trèpharchie** (Aristois) : "Croissance Souveraine"
  - **Mignusophie** (Griscribes) : "Sagesse Imbriquée"
  - **Thrasonirie** (Slaadéens) : "Audace Rêveuse"

#### Motivation (+3D)
- **Envers Gens ou Choses** : Attitude personnelle
- 2x8 Motivations (16 total)
- Déterminée par l'Attribut le plus élevé

**Motivations (Gens)** :
- **Réussite** (FOR) : Le Meilleur, La Capabilité, La Libération
- **Liberté** (AGI) : L'Instinct, La Solitude, L'Autonomie
- **Exploration** (DEX) : La Beauté, La Surhumanité, L'Introspection
- **Tolérance** (VIG) : La Transcendance, L'Opportunité, L'Honnêteté
- **Générosité** (EMP) : L'Égalité, La Communauté, Les Biens Communs
- **Sécurité** (PER) : La Fidélité, La Loi, La Hiérarchie
- **Fondation** (CRÉ) : La Vision, La Perpétuation, La Justice
- **Pouvoir** (VOL) : L'Enrichissement, L'Augmentation, L'Ascendance

**Motivations (Choses)** :
- **Guerre** (FOR) : L'Expérimentation, Le Prototype, La Tactique
- **Survie** (AGI) : L'Artisanat, La Préparation, La Retraite
- **Artifice** (DEX) : La Manipulation, La Divination, Le Mysticisme
- **Labeur** (VIG) : L'Enchantement, La Consécration, La Cérémonie
- **Ornement** (EMP) : La Créativité, La Sacralité, L'Élevation
- **Foi** (PER) : L'Illusion, L'Abstraction, La Sublimation
- **Rite** (CRÉ) : L'Interprétation, L'Idée, La Conception
- **Sortilège** (VOL) : L'Observation, Le Théorème, La Stratégie

#### Personnalité (+1D)
- **Réactions à l'univers** : Comportement
- Qualité pour soi, défaut pour les autres
- Déterminée par l'Aptitude la plus élevée

**Personnalités par Aptitude** :
- **Puissance** : Véhément, Imposant, Intrépide
- **Aisance** : Tranquille, Apaisé, Décontracté
- **Précision** : Planificateur, Minutieux, Révérencieux
- **Athlétisme** : Intransigeant, Nonchalant, Complexé
- **Charisme** : Extraverti, Empathique, Social
- **Détection** : Dévot, Respectueux, Humble
- **Réflexion** : Imaginatif, Poétique, Étonnant
- **Domination** : Prestigieux, Digne, Influent

### Utilisation des Valeurs
- **Guérison de Folies** : Via Ressentiment ou Consentiment
- **Décisions** : Guident les choix du personnage
- **Rôle-play** : Définissent le comportement

## Traits

### Concept
- **Traits** : Réactions, désirs, aversions
- Guérissent ou définissent les **Rancœurs**
- Chaque Trait a des Dés associés
- Se marquent à chaque Échec

### Types de Traits

#### Socialité (+6D)
- **Défini par l'Âge et le Sexe**
- Change selon l'étape de vie :
  - **Enfance** : Joueur, Monde Inique, Insignifiant, Chanceux, Vulnérable
  - **Jeunesse** : Féminité, Masculinité, Autre
  - **Vieillesse** : Hôte, Bien commun, Marchandage, Survivant, Expérience, etc.
- **Parentalité** : Après avoir enfanté (+1D par enfant/Naiv d'enfantement)

#### Disposition (+3D)
- **Envers Peuples et Castes**
- De l'adoration (+5Niv/+15Dés) à la haine (-5Niv/-15Dés)
- **Base** : +3D envers son Peuple (si vécu en Ruralité/Isolé)
- Ou envers Peuple haï par le sien
- Ou envers Caste liée à Compétence Familiale

#### Passion (+1D)
- **Objet de la Passion** : Alcool, miel, viande, chariots, amant, animal
- S'exprime quand l'objet se présente
- **Base** : +1D, typiquement lié au Symbole et Peuple

### Niveaux d'Étrangeté Culturelle

#### Proximité
- **Soi-même** : +15D/+5N
- **Famille proche** : +10D/+4N
- **Famille étendue** : +6D/+3N
- **Tribu/Caste** : +3D/+2N
- **Race** : +1D/+1N
- **Peuple** : 0
- **Origine** : -1D/-1N
- **Type** : -3D/-2N
- **Racine** : -6D/-3N
- **Vie** : -10D/-4N
- **Essence** : -15D/-5N

#### Temporelle
- **Présent** : 0
- **Passé** : -1D/-1N
- **Histoire** : -3D/-2N
- **Antiquité** : -6D/-3N
- **Mythe** : -10D/-4N
- **Éternité** : -15D/-5N

### Utilisation des Traits
- **Guérison de Rancœurs** : Via actions appropriées
- **Interactions sociales** : Modifient les jets
- **Rôle-play** : Définissent les réactions

## Ressentiment et Consentiment

### Ressentiment
- **Sacrifier une Valeur** : Réduire ses Dés
- **Guérit les Souffrances** : Selon le type
- **Coût** : Perte permanente de Dés

### Consentiment
- **Gagner des Dés d'Élan** : Vital ou Morbide
- **Guérit les Souffrances** : Selon le type
- **Bénéfice** : Dés modifiant tous les jets et effets Riliques

## Dés d'Élan

### Élan Vital
- **Gagné via Consentiment**
- Modifie positivement tous les jets
- Affecte les effets Riliques

### Élan Morbide
- **Gagné via Ressentiment**
- Modifie négativement tous les jets
- Affecte les effets Riliques
