# Monde – Iäoduneï

*Sources : Livre p. 9–10, 23, 140–144 ; cohérent avec 12_Peuples_Races et LORE.*

**Orthographe livre** : **Iäoduneï** (Pays d’Iäoduneï, région aux dunes enneigées, eaux brûlantes, montagnes centrales). Parfois « Iâodunaï » en variante.

---

## Région de Jeu

### Iäoduneï
- **Population** : ~200'000 sur 232'000 km²
- **Localité principale** : Mangroves d'Hylothermes
- **Caractéristiques** : Dunes enneigées, eaux brûlantes, montagnes centrales

### Lieux Importants
- **Dla'Adur** : Plus grand peuplement Nomachome
- **Lumioleï** : Second lieu majeur
- **Monts Ajolum** : Montagnes centrales
- **Déserts Soufflés** : Région désertique
- **Grands Lacs Thermaux** : Tuhial, Eos, Daghol

## Factions Principales

### Slaadéens
- **N'la-Fatch** : Haute-Famille artistique (Felichrem, Iossolam)
- **Souk'mans** : Infiltration discrète ("Oigothyriser")
- **Braisois** : Sédentaires, culte Heptahalie

### Autres Factions
- **Rophanes** : Microcité-états
- **Daïnumes Aristois** : Communautés variées
- **Aïozhan & Zoluzhan Gomb** : Syndicats
- **Ennakleis** : Illégalité
- **Jademarche** : Causes impopulaires

## Conflits

### Économiques
- **Chasseurs-cueilleurs** (Slaadéens) vs **Pastoralisme** (Aristois) vs **Agriculture** (Ylfes des lacs)

### Culturels
- **Nomades** vs **Sédentaires / Semi-sédentaires** (ex. Slaadéens : majorité nomade en opposition à semi-sédentaire ; Méridiens, Aristois, Navillis plutôt nomades ; Ylfes des lacs semi-sédentaires) (Livre p. 141).
- **Techno-traditionalisme** : les Motivations opposent l’ordre des **Gens** (société, coutumes) et des **Choses** (nature, technologie) ; le setting mêle traditions et techniques (Livre p. 9–10, 129).

### Religieux
- **Heptahalie** vs **Oratoire Kaïaste** vs **Nomê Aristois** vs **Hahalumullio Ylfe**

## Glossaire - Mangroves d'Hylothermes

### Général
- **Nomachome** : Caravane-villages nomades
- **Hylotherme** : Arbres mangroviens aux feuilles carmines

### Faune
- **B'dyiorlyr** : Créature ailée fouisseuse
- **Arvadoy** : Bétail géant à queue préhensile
- **Luicéphale** : Créatures lumineuses pour lanternes
- **Ipslithes** : Vers-cristaux souterrains
- **Zoion** : Organismes microscopiques

### Matériaux
- **Driliz** : Alliage Éore+Avathon (vert viride)
- **Chraméon** : Alliage Éore+Pseudiris (solaire/rosé)
- **Éore** : Métal bleu de nuit
- **Avathon** : Poussière bleu-roi
- **Pseudiris** : Animalcule minéral liquide
- **Pittlithe** : Charbon aqueux brillant
- **Faéine** : Plante-métal dorée
- **Bryocarte** : Papier algueux

### Produits
- **Mycolithe** : Roche champignonneuse comestible
- **Cloche-fruit** : Fruits des hylothermes
- **Fythymel** : Miel-algue thermal
- **Bryophloï** : Écorce-mousseuse

## Yul - Exemple de Lieu

### Description
- **Type** : Chefferie Nomade NdG 6 (~1000 habitants)
- **Peuple** : Slaadéens
- **Faction** : N'la-Fatch (Ouvriers)
- **Atout** : Caves aux Zoions (5dD)
- **Localisation** : Presqu'entièrement sous terre, pieds des monts Ajolum

### Système de Rencontres
- Table de terrain (-5 à +5)
- Factions rencontrées selon le résultat
- Opportunités et dangers variables
