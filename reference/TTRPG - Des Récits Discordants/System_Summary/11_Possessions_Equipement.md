# Possessions et Équipement

## Devises

### Monnaie
- **cO (coins)** : Unité de base
- **Valeur** : Variable selon les régions
- **Formes** : Pièces, bijoux, bétail, matières premières

### Échange
- **Bétail** : 1 bétail = 100 cO (approximatif)
- **Matières premières** : Valeur variable
- **Troc** : Échange direct de biens

## Qualité, Rareté, Utilité

### Qualité
- **Pauvre** : -1 Dé aux jets
- **Normale** : Pas de modificateur
- **Bonne** : +1 Dé aux jets
- **Excellente** : +2 Dés aux jets
- **Légendaire** : +3+ Dés aux jets

### Rareté
- **Commune** : Facile à trouver
- **Rare** : Difficile à trouver
- **Très rare** : Très difficile
- **Unique** : Un seul exemplaire

### Utilité
- **Inutile** : Aucun avantage
- **Utile** : Avantage mineur
- **Très utile** : Avantage significatif
- **Essentiel** : Nécessaire pour certaines actions

## Marché

### Achat
- **Prix de base** : Selon Qualité/Rareté/Utilité
- **Négociation** : Jet de [Négociation]

### Vente
- **Valeur** : Moins que l'achat (marge)
- **Négociation** : Jet de [Négociation]
- **Relations** : Modifient les prix
- **Demande** : Variable selon la région
