## Les Concepts Essentiels

### 1. Les Dés (dD)
- **3 faces** : `+`, `-`, `0`
- **Résultat** : Nombre de `+` moins nombre de `-`
- **Exemple** : `++-0-` = 2 - 1 = **+1**

### 2. Les Épreuves
- **Épreuve Possible** : Action avec chance d'échec
- **Épreuve Impossible** : Action sans chance (sauf critique)
- **Niv d'Épreuve** : Difficulté (-5 à +10+)

### 3. Les Jets
**Base** : Niv d'Aptitude + Niv de Compétence + Dés de Compétence
**Modificateurs** : Situation, équipement, Souffrances
**Succès** : Résultat ≥ Niv d'Épreuve

### 4. L'Expérience
- **Marques** : 1 par Échec (5 si Échec Critique)
- **10 Marques** = Compétence Éprouvée
- **Réalisation** : +1 Dé, Marques gratuites, +1 Attribut

### 5. Les Souffrances
- **8 types** : Blessures, Fatigues, Entraves, Disettes, Addictions, Maladies, Folies, Rancœurs
- **Dés Négatifs** : Appliqués aux jets liés
- **Seuils** : 3-5 DS (Passagère), 6-9 (Durable), 10-11 (Permanente), 15-20 (Fatale)
- **Rage** : 10+ Souffrances totales
- **Évanouissement** : 15+ Souffrances totales
- **Vaincu** : 21+ Souffrances

## Création de Personnage Simplifiée

### Étape 1 : Origine, Peuple, Race (5 min)
Choisissez :
- **Origine** : Yômmes, Yôrres, ou Bêstres
- **Peuple** : Parmi ceux de l'Origine
- **Race** : Parmi celles du Peuple

### Étape 2 : Attributs (5 min)
- Prenez les Attributs de base du Peuple/Race
- Ajoutez +2, +1, 0, 0, 0, 0, -1, -2 où vous voulez

### Étape 3 : Compétences (10 min)
- **3 Compétences Populaires** : +1D chacune
- **2 Compétences Raciales** : +2D chacune
- **1 Compétence Familiale** : +3D

### Étape 4 : Symbole (5 min)
- Votre Attribut le plus élevé → Symbole
- 3 Compétences avec 3 Marques Éternelles

### Étape 5 : Valeurs et Traits (5 min)
- **1 Moralité** : +6D (héritée du Peuple)
- **1 Motivation** : +3D (envers Gens ou Choses)
- **1 Personnalité** : +1D (liée à l'Aptitude)
- **1 Socialité** : +6D (Enfance → Sexe → Vieillesse)

### Étape 6 : Ressources (5 min)
- **1000 cO** : Équipement de base
- **1 Titre** : Niv de Gouvernance et Dés d'Influence
- **Langues** : Parlé Exercé, Écrit Essentiel, Signé Enfantin

**Total : ~40 minutes pour un personnage jouable**

## Première Session

### Avant la Session
1. Créer un personnage (méthode Amnésique ou Complète)
2. Lire le Quick Start
3. Préparer des questions

### Pendant la Session
1. **Écouter** : L'Éveilleur explique le contexte
2. **Poser des questions** : Mieux vaut demander que supposer
3. **Tester** : Essayez différentes actions
4. **Apprendre** : Chaque échec vous apprend quelque chose

### Après la Session
1. **Noter** : Ce que vous avez appris
2. **Compléter** : Votre personnage (si méthode Amnésique)
3. **Réfléchir** : Comment vos Souffrances ont affecté le jeu

## Questions Fréquentes

### "C'est trop complexe !"
→ Commencez par le Quick Start. Le reste se découvre progressivement.

### "Je ne comprends pas les Souffrances"
→ Lisez `05_Souffrances_Guérison.md`. Commencez par les bases, ignorez les détails.

### "Comment créer un personnage rapidement ?"
→ Utilisez la méthode Amnésique. Créez le minimum, découvrez le reste en jouant.

### "Où sont les sorts de magie ?"
→ Il n'y en a pas. La Rilie fonctionne différemment. Lisez `07_Rilie_Magie.md`.

### "Le combat est compliqué"
→ Commencez simple. Ignorez les détails, concentrez-vous sur : Proaction → Réaction.

### "Je veux tout comprendre maintenant"
→ C'est normal ! Mais prenez votre temps. Le système est dense. Lisez progressivement.

## Conseils pour Nouveaux Joueurs

1. **Ne lisez pas tout d'un coup** : C'est écrasant. Lisez progressivement.

2. **Commencez simple** : Personnage simple, situation simple, règles simples.

3. **Posez des questions** : Mieux vaut demander que supposer.

4. **Acceptez l'échec** : L'échec est une partie du système, pas un problème.

5. **Explorez progressivement** : Découvrez les mécaniques une par une.

6. **Jouez avec des gens patients** : Le système nécessite de la patience.

7. **Amusez-vous** : Malgré la complexité, c'est un jeu. Amusez-vous !
