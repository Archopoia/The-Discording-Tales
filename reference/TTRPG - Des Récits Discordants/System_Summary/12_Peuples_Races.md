# Peuples et Races

## Origines

### Yômmes (Humanoïdes Standards)
- **Peuples** : Aristois, Griscribes, Navillis, Méridiens
- **Caractéristiques** : Variées, généralement équilibrées
- **Espérance de vie** : 40-80 ans (selon Peuple)

### Yôrres (Elfes et Similaires)
- **Peuples** : Hauts Ylfes, Ylfes pâles, Ylfes des lacs, Iqqars
- **Caractéristiques** : Longévité, souvent liés à la nature
- **Espérance de vie** : 100-200+ ans

### Bêstres (Créatures Diverses)
- **Peuples** : Slaadéens, Tchalkchaïs
- **Caractéristiques** : Très variées, souvent spécialisées
- **Espérance de vie** : Variable (2-115 ans selon Peuple)

## Yômmes

### Aristois
- **Moralité** : Trèpharchie ("Croissance Souveraine")
- **Attributs** : FOR+, EMP+, VIG+
- **Races** : Monte-Foudres, Fleuris, Des Souffles, Ambrés
- **Relations** : Amis avec Griscribes, Confiants avec Navillis

### Griscribes
- **Moralité** : Mignusophie ("Sagesse Imbriquée")
- **Attributs** : VOL+, CRÉ+, DEX+
- **Races** : Arboricoles, Saute-Brumes, Limnicoles, D'Eau
- **Relations** : Amis avec Navillis, Confiants avec Aristois

### Navillis
- **Moralité** : Olbôphélie ("Félicité Dévouée")
- **Attributs** : VIG+, VOL+, EMP+
- **Races** : Carmin, Pétricole, Perce-Vague
- **Relations** : Alliés avec Griscribes, Confiants avec Aristois

### Méridiens
- **Moralité** : Etéathlie ("Authenticité Compétitrice")
- **Attributs** : DEX+, VOL+, FOR+
- **Races** : Sylvains, Des Monts, Marche-Rêves
- **Relations** : Hostiles avec Navillis, Méfiants avec plusieurs

## Yôrres

### Hauts Ylfes
- **Moralité** : Edapoèsie ("Compétence Créative")
- **Attributs** : FOR+, CRÉ+, EMP+
- **Races** : De Lunes, Des Plaines
- **Relations** : Hostiles avec plusieurs, Amis avec Iqqars

### Ylfes pâles
- **Moralité** : Hiéragogie ("Direction Sacrée")
- **Attributs** : DEX+, CRÉ+, EMP+
- **Relations** : Hostiles avec plusieurs, Ennemis avec Hauts Ylfes

### Ylfes des lacs
- **Moralité** : Mellophrônie ("Volition Ingénieuse")
- **Attributs** : VOL+, FOR+, CRÉ+
- **Races** : Mâche-Troncs, Danse-Feuilles, Troglodytes
- **Relations** : Hostiles avec plusieurs, Ennemis avec Tchalkchaïs

### Iqqars
- **Moralité** : Égonexie ("Intimisme Extrinsèque")
- **Attributs** : FOR+, CRÉ+, AGI+
- **Races** : Hauts, Bas
- **Relations** : Harmonie avec Hauts Ylfes, Accueil avec plusieurs

## Bêstres

### Slaadéens
- **Moralité** : Thrasonirie ("Audace Rêveuse")
- **Attributs** : EMP+, DEX+, EMP+
- **Races** : Sydru-kal, Wijriol, Hlabayan, Felichrem, Iossolam
- **Espérance de vie** : 2-115 ans (très variable)
- **Relations** : Neutres avec plusieurs, Confiants avec Hauts Ylfes

### Tchalkchaïs
- **Moralité** : Echothysie ("Altruisme Sacrifié")
- **Attributs** : FOR+, EMP+, VOL+, VIG+
- **Races** : Elesihin, Ordol, Îohadam, Nuôtarjoy, Ikrayad
- **Espérance de vie** : 10-27 ans (courte)
- **Relations** : Accueil avec Navillis/Iqqars, Hostiles avec Ylfes

## Compétences Sociétales, Populaires et Raciales

### Compétences Sociétales
- **3 Compétences** communes à tout le Peuple
- Toujours acquises

### Compétences Populaires
- **1 Compétence** liée au Peuple
- +1D lors de la création

### Compétences Raciales
- **1 Compétence** liée à la Race
- +2D lors de la création

## Relations Inter-Peuples

### Calcul de Disposition
- **Moyenne** des différences d'Attributs
- **-Niv de Parenté**
- **+|-1** selon Empathie du Peuple
- **+3** si Humanoïde
- **Modificateurs** : Passé, événements, aléas

### Niveaux de Disposition
- **Harmonie +5** : Alliance profonde
- **Ami +3** : Amitié forte
- **Allié +4** : Alliance
- **Confiant +2** : Confiance
- **Accueil +1** : Accueil favorable
- **Neutre 0** : Indifférence
- **Méfiant -1** : Méfiance
- **Hostile -2** : Hostilité
- **Ennemi -3** : Inimitié
- **Rival -4** : Rivalité intense

## Métissages

### Règles
- **Entre Origines** : Impossible
- **Même Peuple, Races différentes** : Souvent possible
- **Même Origine, Peuples différents** : Rare, souvent infertile
- **Bêstres/Iqqars** : Trop divers ou isolés

### Héritage
- **Yômmes** : Traits maternels
- **Yôrres** : Traits paternels
