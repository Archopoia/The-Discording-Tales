# Dynamiques de Groupe

## Jet d'Ambiance

### Concept
- **Ambiance** : Cohésion du groupe
- Affecte les actions collectives
- Se modifie selon les situations

### Déclenchement
- Situations significatives
- Séparations du groupe
- Conflits internes
- Succès/échecs collectifs

### Calcul
- **Jet d'Ambiance** : Jet contre Épreuve
- **Base** : Niv de Gouvernance + Niv d'Ambiance de la situation
- **Succès** : +Dés à l'Ambiance du groupe
- **Échec** : -Dés à l'Ambiance du groupe

### Pool d'Ambiance
- Dés disponibles pour le groupe
- Utilisés dans les Jets de Groupe
- Retour progressif à neutre

## Jet de Groupe

### Concept
- Action collective
- Jets individuels modifiés par l'Ambiance
- Résultats moyennés

### Mécanisme
1. Chaque membre fait un jet individuel
2. Modifié par l'Ambiance du groupe
3. Résultats moyennés
4. Succès si moyenne ≥ Niv d'Épreuve

### Utilisation de l'Ambiance
- **+X Dés** : Ajoutés à chaque jet individuel
- **-X Dés** : Retirés de chaque jet individuel
- Basé sur le pool d'Ambiance actuel

## Jet d'Audace

### Concept
- Un individu tente une action de groupe
- Pénalités importantes
- Récompenses potentielles plus élevées

### Mécanisme
- **Jet individuel** avec pénalités
- **Pénalité** : -1 Dé par membre du groupe (au-delà de 1)
- **Récompense** : Multipliée si succès

### Utilisation
- Quand le groupe ne peut/veut pas agir ensemble
- Action risquée mais nécessaire
- Démontrer sa valeur

## Retour à la Normale

### Récupération d'Ambiance
- Retour progressif à neutre
- Vitesse selon les événements
- Peut être accéléré par actions positives

### Événements Modifiant l'Ambiance
- **Positifs** : Succès collectifs, coopération, entraide
- **Négatifs** : Échecs, conflits, trahisons, séparations

## Niv de Gouvernance

### Concept
- Capacité d'organisation du groupe
- Détermine la base du Jet d'Ambiance
- Variable selon la structure du groupe

### Types de Groupes
- **Bande** : Niv faible
- **Tribu** : Niv modéré
- **Chefferie** : Niv élevé
- **État** : Niv très élevé

## Coopération

### Actions Coordonnées
- Plusieurs personnages agissent ensemble
- Bonus selon la coordination
- Partage des risques/récompenses

### Leadership
- Un personnage dirige l'action
- Utilise ses Compétences de [Commandement]
- Bonus au groupe si succès

## Conflits Internes

### Causes
- Divergences d'opinion
- Compétition pour ressources
- Trahisons
- Incompatibilités de Valeurs/Traits

### Résolution
- **Sociale** : Négociation, compromis
- **Violente** : Combat interne
- **Séparation** : Groupe se divise

### Effets
- Réduction d'Ambiance
- Perte de cohésion
- Possible dissolution du groupe

## Groupes Multiples

### Alliances
- Plusieurs groupes travaillent ensemble
- Ambiance séparée par groupe
- Coordination entre groupes

### Conflits Inter-Groupes
- Compétition entre groupes
- Guerres, rivalités
- Négociations

## Rôles dans le Groupe

### Leader
- Prend les décisions importantes
- Utilise [Commandement]
- Responsable de l'Ambiance

### Suiveur
- Suit les décisions
- Contribue à l'Ambiance
- Peut devenir Leader

### Médiateur
- Résout les conflits
- Utilise [Négociation], [Inspiration]
- Maintient la cohésion

### Spécialiste
- Expert dans un domaine
- Contribue via ses Compétences
- Reconnu pour son expertise
