Lors du dernier Clin d’affilé de vos Proactions déclarées, vous devrez faire votre jet d’Attaque combinant le Niv de Puissance , les Dés de [ Frappe ] ( en mêlée ) ou [ Tir ] ( à distance ), et le Niv d’Attaque ( si vous attaquez priorisant les dégâts ) ou de Manœuvre ( si vous priorisez le contrôle via un botte ou une lutte ) de l’arme utilisée.

- Puis, comparez votre résultat au jet d’Attaque avec le Blocage ( Puissance [ Neutraliser:( Dés ) Armé ] + Parade d’arme) , la Parade ( Précision [ Manier:( Dés )Armé ] + Parade d’arme) ou l’Esquive ( Aisance [ Réagir:Esquive ] ) de la cible. SI vous y obtenez au moins 1 Succès ou plus, vous pourrez les dépenser afin d’influencer votre Attaque.
- SI votre Portée d’arme est inférieure à 1 ( une case de distance touchable ), il vous faudra d’abord vous Déployer dans la zone de l’ennemi ( une ou deux fois d’un coup, afin d’augmenter votre Portée de + 0 . 5 voir + 1 ).
- FRAPPER votre cible d’une des trois différentes façons ( qu’une seule façon par Attaque ).
- ( 1 S ) Frontale ( l’action de base et nécessaire afin d’infliger des Dégâts ).
- ( 2 S ) Latérale ( infliger plus de dégâts par la portée etl’inertie de l’arme, contre quelques Clins d’affilé supplémentaire afin de récupérer le coup ).
- ( 3 S ) Glissement ( infligeant au prochain Clin: des Dégâts si la face de l’arme est Tranchante, des Manœuvres si elle est Contondante, et l’un ou l’autre si elle est Perforante en fonction de si vous avez pénétré les défenses de la cible ou non ). On peut combiner un Glissement avec une Frappe Frontale ou Latérale lors d’une même Attaque ( voir ci - dessous ).
- AUGMENTER votre type de Frappe infligée ( coûtant respectivement + 1 S, + 2 S, ou + 3 S ).
- TRAVERSER plusieurs cibles à la fois à travers votre même attaque ( dépensez X + 1 | 2 | 3 | 4 | 5 |etc. Succès comme un résultat de jet d’Attaque X sur + 1 | 2 | 3 | 4 | 5 ème |etc. Cibles adjacentes à la précédente et à votre portée ).
