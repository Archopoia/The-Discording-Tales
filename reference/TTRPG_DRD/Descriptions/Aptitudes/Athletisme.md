# Athlétisme

**Attributs :** VIG +3, FOR +2, AGI +1  
**Type de Conflit :** Prouesse (Outrepassement physique)

---

## Description `[ADAPTÉ]`

L'Athlétisme, c'est l'aptitude d'un être à s'exercer physiquement au travers de la course, l'escalade, le jet, la nage, le saut, le sprint, le transport et tout travail physique sur court et long terme. Votre Athlétisme définit votre maîtrise de ces éléments ainsi que la facilité de les pratiquer.

## Type de Conflit : Prouesse

**Approche :** Créer une autre voie, par le possible

- **Objet (gardé) :** Prendre de force l'Objet sécurisé
- **Obstacle (gardien) :** Esquiver ou Surmonter l'Obstacle

Exploit physique et endurance. On utilise la vigueur et les capacités athlétiques pour surmonter des défis physiques. On ne combat pas directement, mais on affronte l'environnement lui-même : escalader, sauter, porter, résister aux éléments.

**Exemples :** Escalader un mur, traverser une rivière à la nage, soulever une lourde pierre, résister à une tempête, sauter un précipice.

## Extrêmes de l'Aptitude

**(-5 et moins) :**
- Aucune MOBILITÉ possible
- Paralysie basse généralisée empêchant tout mouvement, corps immobile, affaissé, peau pâle et froide

**(+5 et plus) :**
- Immunisé à l'ENCOMBREMENT
- La charge reste la même et il y a toujours une limite, mais elle n'a aucun effet (aucune Souffrance n'en est générée)

# Action : Traverser

L'action de se déplacer à travers différents milieux.

---

## [Pas]

### Description `[ADAPTÉ]`
Marche, course, jogging, rampe, marathon. Aussi le vol pour ceux dans les airs ou le déplacement sans intermédiaires (espace).

### Découvertes
1. Divisez par deux les échecs de fatigue liée à cette Compétence
2. Charger/Courir/Ramper/etc. n'a aucun effet négatif sur toute action
3. Immunisé à toute charge portée (aucun de ces dés négatifs sur cette Compétence)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Ramper** | Se déplacer à plat ventre pour éviter les obstacles et ennemis |
| **Marcher** | Se déplacer en marchant de manière stable pour parcourir de longues distances |
| **Courir** | Se déplacer rapidement pour atteindre des destinations ou poursuivre |
| **Charger** | Se déplacer à vitesse maximale pour de brèves périodes d'urgence |
| **Pédaler** | Se déplacer en utilisant des mécanismes rotatifs et pédales |

---

## [Grimpe]

### Description `[ADAPTÉ]`
Grimpe dans les arbres, montagnes, glaciers ou autres surfaces plus ou moins lisses, collantes, voire sur des êtres eux-mêmes.

### Découvertes
1. Immunisé aux effets de parois (mouvante (créature), glissante (mouillé), etc.)
2. Le fait d'être en train de grimper n'a aucun effet négatif sur toute action
3. Immunisé à toute charge portée (aucun de ces dés négatifs sur cette Compétence)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Montagnard** | Escalade en montagne |
| **Glaciaire** | Escalade sur glace |
| **Descendant** | Descente contrôlée |
| **en Rappel** | Descente avec corde |
| **sur Créature** | Grimper sur un être vivant |

---

2. Le fait d'être en train de nager n'a aucun effet négatif sur toute action
3. Immunisé à toute charge portée (aucun de ces dés négatifs sur cette Compétence)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Plongeant** | Plongée sous-marine |
| **Contre-courant** | Nager contre le courant |
| **de Compétition** | Nage rapide et technique |
| **Flotter surplace** | Rester immobile à la surface |

# Action : Efforcer

L'action de fournir un effort physique intense.

---

## [Port]

### Description `[ADAPTÉ]`
Porter, équiper, pousser, tirer, soulever, appuyer, ouvrir, fermer, jeter tout poids ou chose avec force.

### Découvertes
1. Divisez par deux les échecs de fatigue liée à cette Compétence
2. Votre équipement est toujours équipé dès que la situation le requiert
3. Tous les équipements portés ne comptent pas comme dés de charge

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Tirer & Pousser** | Manier les objets lourds avec force contrôlée |
| **Soulever & Ouvrir** | Soulever et ouvrir les objets pesants avec efficacité maximale |
| **Porter** | Transporter des charges lourdes sur de longues distances |
| **Lancer** | Jeter des objets avec précision et force |
| **Supporter (Équiper)** | Porter des équipements lourds confortablement, minimiser les restrictions de mouvements |

---

## [Saut]

### Description `[ADAPTÉ]`
Saut en hauteur, en longueur, à pieds joints, etc.

### Découvertes
1. Divisez par deux les échecs de fatigue liée à cette Compétence
2. Doublez vos succès à un seul jet lié (1x / NdT) afin de survivre, s'enfuir, etc.
3. Immunisé à toute charge portée (aucun de ces dés négatifs sur cette Compétence)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Sans élan** | Effectuer des sauts sans départ, sans courir |
| **Précis** | Faire des sauts précis et contrôlés, atterrir exactement où souhaité |
| **en Longueur** | Couvrir une longue distance en un seul saut |
| **en Hauteur** | Sauter haut dans les airs |
| **de Paroi** | Sauter des murs pour atteindre de nouvelles hauteurs |
| **à la Perche** | Saut en prenant appui sur une perche |

---

## [Fouissage]

### Description `[ADAPTÉ]`
Creuser et se déplacer dans les sols, ou dans les liquides trop denses pour y nager normalement.

### Découvertes
1. Tout "moyen" creuse un milieu au-delà de votre utilité (mains = sable → terre)
2. Tout fouissage donné prend deux fois moins de temps
3. Vous contrôlez si et quand vos tunnels s'effondrent (sauf situation externe)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Viscosité & Liquides** | Se déplacer dans des liquides épais |
| **Sables & Granulaires** | Creuser dans le sable et matériaux granulaires |
| **Terres & Gravats** | Creuser dans la terre et les gravats |
| **Roches & Solides** | Creuser dans la roche et matériaux solides |

## [Chevauchement]

### Description `[ADAPTÉ]`
Chevauchement de toute bête afin de la faire se déplacer, se battre, manœuvrer, afin de s'y battre soi-même tout en la montant.

### Découvertes
1. Peut monter une créature (ou un véhicule) lorsqu'étant à vitesse maximale
2. Le fait d'être en chevauchement n'a aucun effet négatif sur toute action
3. Immunisé au démontement (tomber de sa monture/sortir de votre véhicule)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Montée en selle** | Monter rapidement sur une monture |
| **Déplacement monté** | Se déplacer efficacement sur une monture |
| **Manœuvres montées** | Effectuer des manœuvres complexes à dos de monture |
| **Agissement monté** | Agir (combattre, etc.) tout en étant monté |
