# Puissance

**Attributs :** FOR +3, AGI +2, DEX +1  
**Type de Conflit :** Bataille (Compétition frontale)

---

## Description `[ADAPTÉ]`

L'essentiel est que la Puissance au Combat est l'expérience et capacité actuelle en la matière, afin de combattre au corps à corps ou à distance, armé ou non. Certaines armes requièrent différents muscles, comme un arc utilisant ceux du dos ou une rapière ceux de l'avant-bras. D'autres Aptitudes peuvent rentrer en compte lors d'une bataille, mais ici c'est le lien direct avec le combat et ses styles dont il est question.

## Type de Conflit : Bataille

**Approche :** Traverser sa voie, frontalement

- **Objet (gardé) :** Tuer ou Détruire et prendre l'Objet
- **Obstacle (gardien) :** Tuer ou Détruire l'Obstacle et passer

Confrontation directe et violente. On affronte l'opposition de face, en utilisant la force brute et les compétences de combat. C'est l'approche la plus directe : vaincre pour obtenir ou passer.

**Exemples :** Combat au corps à corps, bataille rangée, destruction d'une barricade, forcer une porte verrouillée.

## Extrêmes de l'Aptitude

**(-5 et moins) :**
- Aucun COMBAT possible
- Incapacité physique généralisée, corps visiblement fragile, émacié, frêle, articulations gonflées

**(+5 et plus) :**
- Réagir et surréagir N'ÉBRANLE PAS
- Yeux déterminés et sauvages, posture prédatrice et dominante, muscles sculptés

# Action : Frapper

L'action de porter des coups offensifs sur un adversaire.

---

## [Armé]

### Description `[ADAPTÉ]`
Toutes les armes de mêlée portées à la main. Inclut les armes de poigne (sur les mains/doigts), d'antipôle (tête lourde comme haches/masses), de parade (boucliers), de garde (épées), équilibrées (lances/bâtons) et flexibles (fléaux/fouets).

### Découvertes
1. Votre arme est toujours équipée et dégainée dès que la situation le requiert
2. Toute attaque Armée réussie octroie aussi +1 Succès de Botte
3. Si visant la même cible et après une réussite, ajouter votre Niv à l'attaque

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Arme de Poigne** | Armes sur les mains, doigts, ou remplaçant le membre - poings nus inclus |
| **Arme d'Antipôle** | Armes à tête lourde (haches, masses, marteaux, hallebardes) |
| **Arme de Parade** | Armes défensives (boucliers, targes, écus, dagues de parade) |
| **Arme de Garde** | `[MANQUANT]` |
| **Armes Équilibrées** | Lames (épées, rapières), bâtons, armes au centre de gravité bas |
| **Armes Flexibles** | Armes à corps flexible (chaînes, fléaux, fouets, cordes) |

---

## [Désarmé]

### Description `[ADAPTÉ]`
Tous les coups portés sans arme : poings, pieds, coudes, genoux, tête, corps entier. Inclut les techniques de combat rapproché et les coups sans espace pour se déplacer.

### Découvertes
1. Toute attaque Désarmée échouée peut en même temps Saisir l'ennemi (à +1)
2. Toute attaque Désarmée réussie octroie +1 Succès de Ruse
3. Pour chaque attaque (Armé/Désarmé/Improvisé), +1 coup désarmé (à tester)

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Coup sans espace** | Coups impulsifs sans élan, utilisant le corps entier dans un espace restreint |
| **Poings** | Coups de poing de toutes sortes |
| **Pieds** | Coups de pied, de talon, balayages |
| **Coude** | Coups de coude |
| **Genou** | Coups de genou |
| **Corps** | Coups de tête, d'épaule, charges corporelles |

---

## [Improvisé]

### Description `[ADAPTÉ]`
Se battre avec tout ce qui n'est pas une arme conventionnelle : bancs, chopes de bière, bouteilles, outils, mobilier, etc.

### Découvertes
1. En entrant dans un lieu, vous y voyez tous les pièges/armes "improvisables"
2. Tout objet, si justifié, peut améliorer une autre action (glisser, grimper, etc.)
3. Chaque nouvelle utilisation d'une arme improvisée surprend l'ennemi

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Arme à coupures** | Objets tranchants improvisés (verre brisé, etc.) |
| **Arme à pieds** | Objets utilisés avec les pieds |
| **Arme rondes** | Objets ronds ou sphériques |
| **Arme de mains** | Objets tenus en main |
| **Arme de paume** | Objets plats tenus dans la paume |
| **Arme de lien** | Cordes, chaînes, liens improvisés |
| **Jet d'arme improvisée** | Lancer des objets improvisés |

# Action : Neutraliser

L'action de contrôler, immobiliser ou désavantager un adversaire sans nécessairement le blesser.

---

## [Lutte]

### Description `[ADAPTÉ]`
Toutes les techniques des manœuvres de lutte armée ou non : saisies, bousculades, mises à terre, immobilisations, soumissions, désarmements, destruction d'arme, battements, etc.

### Découvertes
1. Toute Lutte réussie octroie aussi +1 succès de Botte ou Ruse
2. Toute Lutte échouée annule tout échec au-delà du premier
3. Immunisé face aux Luttes ennemies

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Saisie** | Attraper et maintenir un adversaire ou ses membres |
| **Bousculade** | Pousser, repousser, déséquilibrer |
| **Mise à Terre** | Faire tomber l'adversaire au sol |
| **Projection** | Lancer l'adversaire par-dessus soi ou à distance |
| **Soumission** | Techniques d'immobilisation douloureuses forçant l'abandon |

---

## [Bottes]

### Description `[ADAPTÉ]`
Techniques spéciales de combat utilisant l'arme de manière particulière, au-delà des simples attaques.

### Découvertes
1. Toute Botte réussie octroie aussi +1 succès de Lutte ou Ruse
2. Toute Botte échouée annule tout échec au-delà du premier
3. Immunisé face aux Bottes ennemies

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Bloquer** | Arrêter une attaque avec son arme |
| **Agrippement** | Utiliser l'arme pour accrocher l'adversaire ou son équipement |
| **Entravement** | Gêner les mouvements de l'adversaire avec l'arme |
| **Désarmement** | Faire lâcher son arme à l'adversaire |
| **Prise d'arme** | S'emparer de l'arme de l'adversaire |
| **Retournement d'arme** | Utiliser l'arme de l'adversaire contre lui |

---

## [Ruses]

### Description `[ADAPTÉ]`
Feintes, tromperies et tactiques de combat pour gagner l'avantage.

### Découvertes
1. Toute Ruse réussie octroie aussi +1 succès de Lutte ou Botte
2. Toute Ruse échouée annule tout échec au-delà du premier
3. Immunisé face aux Ruses ennemies

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Enchaînement** | Bonus en enchaînant les actions d'assaut |
| **Feinter** | Induire en erreur par de faux mouvements |
| **Contre** | Répondre à une attaque par une attaque |
| **Hébétement** | Étourdir ou déstabiliser mentalement l'adversaire |
| **Essouffler** | Fatiguer l'adversaire par le combat |
| **Battement** | Frapper l'arme adverse pour la dévier |
| **Destruction** | Abîmer ou briser l'arme/équipement ennemi |
| **Postures** | Positions de combat avantageuses |
| **Prises d'arme** | Techniques pour saisir une arme en combat |

# Action : Tirer

L'action d'attaquer à distance en projetant des munitions.

---

## [Bandé]

### Description `[ADAPTÉ]`
Toutes les armes à distance bandées à la main et tirant des billes ou flèches : arcs courts, longs, réfléchis, pendules à répétition, etc.

### Découvertes
1. Votre arme est toujours équipée et préparée dès que la situation le requiert
2. Toute attaque Bandée réussie octroie aussi +1 Succès de Visée & Tir
3. Si visant la même cible et après une réussite, ajouter votre Niv à l'attaque

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Encordage** | Mettre la corde rapidement |
| **Surbandé** | Bander l'arc au-delà de sa limite pour plus de puissance |
| **Tirs Courbés** | Faire courber la flèche pour éviter des obstacles |
| **Tirs multiples** | Tirer plusieurs flèches simultanément |

---

## [Propulsé]

### Description `[ADAPTÉ]`
Toutes les armes à distance utilisant un mécanisme de propulsion : frondes, propulseurs, lance-pierres.

### Découvertes
1. Votre arme est toujours équipée et préparée dès que la situation le requiert
2. Toute attaque Propulsée réussie octroie aussi +1 Succès de Visée & Tir
3. Si visant la même cible et après une réussite, ajouter votre Niv à l'attaque

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **Tirs Rapprochés** | Tournoyer verticalement pour tirer à courte distance |
| **Tirs Longue Distance** | Tournoyer latéralement pour tirer à longue distance |
| **Tirs Imprévisibles** | Tournoyer en figure de 8 pour que les cibles ne sachent pas qui est visé |
| **Tirs sur 360** | Tournoyer au-dessus de soi pour tirer dans n'importe quelle direction |

---

## [Jeté]

### Description `[ADAPTÉ]`
Toutes les armes jetées directement de la main : couteaux, étoiles, haches, javelots de lancer.

### Découvertes
1. Jeter plus de choses à la fois ne cause aucun malus
2. Toute attaque Jetée réussie octroie aussi +1 Succès de Visée & Tir
3. Divisez par deux les échecs d'un rattrapage d'une chose tombante/jetée/etc.

### Maîtrises

| Maîtrise | Description |
|----------|-------------|
| **de Paume** | Armes de jet tirées avec les doigts ou la paume (couteaux, étoiles, disques) |
| **à Manche** | Armes de jet ayant un manche (javelots, boomerangs, hachettes) |
| **Rattrapage de jet** | Rattraper des projectiles ou objets en vol |
| **Jets multiples** | Lancer plusieurs projectiles simultanément |
