Il y a 4 types de Conflits ( ayant leurs propres types de règles par type ) liés aux Aptitude ( méca- niques encore à développer ), et qui valorisent toutes les façons d’intéragir avec le monde.

- La Compétition de l’autre où chacun répond de l’autre, ( les Batailles de Puissance et les Débats de Domination ).
- Le Déjouement de l’un fuyant l’autre le poursuivant, les deux utilisant leur environnement à leur propre façon et fin ( les Infiltrations dans l’ Aisance et les Enquêtes à travers la Détection ).
- La Solution d’un obstacle inoffensif, par quiconque et indépendamment ( les Subterfuges par la Précision et les Énigmes par Réflexion ).
- L’Outrepassement d’un obstacle commun, passif mais dangereux ( les Prouesses d’ Athlétisme et les Corruptions de Charisme ).
Lors de tout Conflit, le Révélateur et les Éveilleurs devraient penser aux multiples façons de les résoudre, et / ou de les éviter ( voir le tableau ci - dessous ).

Afin d’obtenir quelque chose, on peut tuer votre gardien, la lui voler, le corrompre ( par l’argent, la séduction, etc. ) qu’il nous le donne, ou encore utiliser un intermédiaire neutre. Afin d’outrepasser un obstacle on peut aussi prendre un autre chemin, tel que se faufiler sans être affecté.
