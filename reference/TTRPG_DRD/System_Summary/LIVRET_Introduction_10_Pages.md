# PAGE 1 : Le Concept

## Des Récits Discordants

**Un jeu où l'échec forge le héros.**

Dans ce jeu, chaque échec vous rapproche de la maîtrise. Vos Souffrances ne sont pas des punitions, mais des moteurs narratifs. À 10+ Souffrances, vous entrez en Rage. À 15+, Évanouissement. Mais chaque Souffrance vous apprend aussi quelque chose.

**Les 3 Principes Fondamentaux :**

1. **L'Échec = Progression** : Chaque échec dans une action possible vous donne une Marque d'expérience. 10 Marques = Réalisation = +1 Dé dans la Compétence.

2. **Souffrances = Moteurs** : Les 8 Souffrances créent des situations, des choix, des dilemmes. Comment gérez-vous votre Rage ? Votre Folie ? Vos Rancœurs ?

3. **Personnalité Mécanique** : Vos Valeurs guérissent vos Folies. Vos Traits guérissent vos Rancœurs. Votre personnalité n'est pas cosmétique, elle est mécanique.

**Pour Jouer :**
- 1 Éveilleur (Maître de Jeu)
- 2-6 Éveilleurs (Joueurs)
- 3 dés à 3 faces par joueur (ou d6 : 1-2 = -, 3-4 = 0, 5-6 = +)

# PAGE 2 : Les Dés

## Les Dés Discordants (dD)

**3 faces** : `+`, `-`, `0`

**Résultat d'un jet** : Nombre de `+` moins nombre de `-`

**Exemples** :
- `++-0-` = 2 - 1 = **+1**
- `+++00` = 3 - 0 = **+3**
- `--000` = 0 - 2 = **-2**
- `00000` = 0 - 0 = **0** (égalité)

**Sans dés spéciaux ?** Utilisez d6 :
- 1-2 = `-`
- 3-4 = `0`
- 5-6 = `+`

## Les Niveaux (Niv)

**Niv d'Attribut** : De -5 (pire) à +5 (meilleur), 0 = moyen

**Niv d'Aptitude** : Calculé à partir de 3 Attributs

**Niv de Compétence** : De 0 à 5 (Néophyte → Maître)

**Niv d'Épreuve** : Difficulté de l'action (-5 à +10+)

# PAGE 3 : Faire un Jet

## Comment Faire un Jet

**Formule** : `Niv + Dés`

**Niv** = Niv d'Aptitude + modificateurs (équipement, situation, etc.)

**Dés** = 5 Dés de Chance + Dés de Compétence + Dés de Potentiel

**Résultat** = Niv + (nombre de `+` - nombre de `-`)

**Succès** : Résultat ≥ Niv d'Épreuve

**Échec** : Résultat < Niv d'Épreuve → +1 Marque d'expérience

## Exemple Simple

**Situation** : Escalader une paroi rocheuse (Niv d'Épreuve +2)

**Personnage** :
- Athlétisme Niv +1
- [Grimpe] Niv 2, +3 Dés
- Équipement : +1 Niv

**Jet** :
- Niv : +1 (Aptitude) + 1 (équipement) = **+2**
- Dés : 5 (Chance) + 3 ([Grimpe]) = **8dD**
- Résultat : +2 + résultat des 8dD

**Si résultat ≥ +2** : Succès, vous escaladez
**Si résultat < +2** : Échec, vous glissez, +1 Marque dans [Grimpe]

## Échecs et Succès Critiques

**Échec Critique** : Tous les dés sont `-` → 5 Marques au lieu de 1

**Succès Critique** : Tous les dés sont `+` → Effet spécial

# PAGE 4 : Création Rapide

## Créer un Personnage (5 minutes)

### 1. Choisir un Peuple
- **Aristois** : Guerriers souverains
- **Griscribes** : Sages imbriqués
- **Navillis** : Navigateurs joyeux
- **Méridiens** : Explorateurs courageux
- **Hauts Ylfes** : Artistes disciplinés
- **Ylfes pâles** : Bergers pragmatiques
- **Ylfes des lacs** : Ingénieux ambitieux
- **Slaadéens** : Visionnaires audacieux
- **Tchalkchaïs** : Altruistes sacrificiels

### 2. Attributs de Base
Chaque Peuple a des Attributs de base. Ajoutez **+2, +1, 0, 0, 0, 0, -1, -2** où vous voulez.

**Les 8 Attributs** :
- **FOR** (Force), **AGI** (Agilité), **DEX** (Dextérité), **VIG** (Vigueur)
- **EMP** (Empathie), **PER** (Perception), **CRÉ** (Créativité), **VOL** (Volonté)

### 3. Aptitudes
Chaque Aptitude = moyenne de 3 Attributs :
- **Puissance** (FOR, AGI, DEX)
- **Aisance** (AGI, DEX, VIG)
- **Précision** (DEX, VIG, EMP)
- **Athlétisme** (VIG, FOR, AGI)
- **Charisme** (EMP, PER, CRÉ)
- **Détection** (PER, CRÉ, VOL)
- **Réflexion** (CRÉ, VOL, FOR)
- **Domination** (VOL, FOR, EMP)

# PAGE 5 : Compétences et Symbole

## Compétences

**Choisissez 3 Compétences** et donnez-leur :
- **+3 Dés** à l'une (Compétence Familiale)
- **+2 Dés** à l'autre (Compétence Raciale)
- **+1 Dés** à la dernière (Compétence Populaire)

**Exemples de Compétences** :
- [Armé], [Désarmé], [Lutte] (Puissance)
- [Esquive], [Danse], [Équilibre] (Aisance)
- [Visée], [Débrouillardise], [Artifices] (Précision)
- [Saut], [Grimpe], [Acrobatie] (Athlétisme)
- [Séduction], [Négociation], [Inspiration] (Charisme)
- [Vision], [Investigation], [Toucher] (Détection)
- [Artisanat], [Médecine], [Ingénierie] (Réflexion)
- [Commandement], [Intimidation], [Apprivoisement] (Domination)

## Le Symbole

Votre **Attribut le plus élevé** détermine votre Symbole (destinée/passion).

**Il donne 3 Compétences** avec **3 Marques Éternelles** chacune (elles comptent pour la progression).

**Exemples** :
- FOR le plus élevé → Symbole Puissance → [Armé], [Désarmé], [Lutte]
- EMP le plus élevé → Symbole Charisme → [Séduction], [Négociation], [Inspiration]

## Valeurs et Traits

**1 Moralité** : +6 Dés (héritée de votre Peuple)
**1 Motivation** : +3 Dés (envers Gens ou Choses)
**1 Socialité** : +6 Dés (selon votre âge)

# PAGE 6 : Les Souffrances

## Les 8 Souffrances

Chaque Souffrance est liée à un Attribut :

1. **Blessures** (FOR) : Dégâts physiques
2. **Fatigues** (AGI) : Épuisement
3. **Entraves** (DEX) : Empêchements
4. **Disettes** (VIG) : Faim, soif
5. **Addictions** (EMP) : Dépendances
6. **Maladies** (PER) : Affections
7. **Folies** (CRÉ) : Troubles mentaux
8. **Rancœurs** (VOL) : Ressentiments

## Comment Obtenir des Souffrances

**Échec dans une Épreuve Possible** → Souffrances selon le type d'échec :
- Échec léger : 1-2 Dés de Souffrance (DS)
- Échec modéré : 3-5 DS → Séquelle Passagère
- Échec sévère : 6-9 DS → Séquelle Durable
- Échec critique : 10-11 DS → Séquelle Permanente

**Exemples** :
- Tomber → Blessures
- Nuit blanche → Fatigues
- Faim → Disettes
- Peur → Folies
- Colère → Rancœurs

## Dés de Souffrance (DS)

Les DS s'appliquent comme **Dés Négatifs** aux jets liés à l'Attribut affecté.

**Exemple** : 5 Blessures (FOR) → -5 Dés à tous les jets de Puissance

# PAGE 7 : Seuils et Rage

## Seuils de Souffrances

**3-5 DS** : Séquelle Passagère (temporaire)
**6-9 DS** : Séquelle Durable (long terme)
**10-11 DS** : Séquelle Permanente (permanente)
**15-20 DS** : Séquelle Fatale (menace de mort)

## Rage et Évanouissement

**10+ Souffrances totales** : **Rage Niv 1**
- Jet 1d6 > Niv de Rage pour agir rationnellement
- Sinon, agissez par instinct (survie)

**15+ Souffrances totales** : **Évanouissement/Démence**
- Échangez 1 Niv de Rage à chaque nouvelle Souffrance
- Jet 1d6 > Évanouissement pour pouvoir Agir

**21+ Souffrances** : **Vaincu**
- Coma ou Folie permanente

## Guérison

**Valeurs** : Guérissent les Folies (CRÉ)
**Traits** : Guérissent les Rancœurs (VOL)
**Traitements** : Guérissent les autres Souffrances
**Temps** : Certaines Souffrances guérissent naturellement

**Résistance** : Utilisez une Compétence Résistante pour réduire les Souffrances reçues.

# PAGE 8 : Combat Simplifié

## Combat en Temps Réel

**Clins** : 1/3 de seconde. Combat en temps réel, pas en tours.

**Clins d'Alerte** : Déterminent qui peut agir de manière proactive.

## Actions au Combat

### Proactions (Actions Offensives)
- **Attaquer** : Frapper, Tirer
- **Déplacer** : Se déplacer
- **Manipuler** : Utiliser un objet

### Réactions (Défensives)
- **Bloquer** : Absorbe les dégâts
- **Parer** : Redirige l'attaque
- **Esquiver** : Évite l'attaque

**Réactions** se font au **dernier Clin** de l'attaque ennemie.

## Niv d'Ébranlement

Accumulé par les Réactions. Réduit vos capacités.

**Exemple** : 3 Ébranlement → -3 Niv à tous vos jets

## Dégâts

**Attaque réussie** → Blessures selon la différence de succès :
- Succès de +1 à +2 : 1-2 DS Blessures
- Succès de +3 à +4 : 3-5 DS Blessures
- Succès de +5+ : 6-9 DS Blessures

# PAGE 9 : Expérience et Progression

## Les Marques

**1 Marque** : Par échec dans une Épreuve Possible
**5 Marques** : Par échec critique

**Marques Éternelles** : Ne comptent pas pour la progression (données par le Symbole)

## Réalisation

**10 Marques** (moins Marques Éternelles) dans une Compétence = **Compétence Éprouvée**

**Réalisation** :
- **+1 Dé** à la Compétence
- **Marques gratuites** ailleurs (1 par Niv de Compétence)
- **+1** à l'Attribut associé

**Exemple** : [Armé] a 10 Marques → Éprouvée → +1 Dé, +1 à FOR

## Distinction

**Distinction** = Somme de tous les Niv d'Attributs

À chaque **+10 Distinction** :
- **+1** à un Attribut
- **Découverte d'Attribut** (capacité spéciale)

## Découvertes

Chaque Compétence a **3 Découvertes** uniques (capacités spéciales) obtenues à certains Niv.

# PAGE 10 : Exemple de Jeu

## Situation

Votre personnage, un **Aristois**, doit négocier avec un marchand pour obtenir des matériaux.

## Le Personnage

- **Empathie +3** (Charisme +3)
- **[Négociation] Niv 2, +2 Dés**
- **8 Folies** actuelles (liées à CRÉ)
- **Valeur Générosité** : +3 Dés

## Le Jet

**Base** : Charisme +3 + [Négociation] 2 + 2 Dés = **5 + 2dD**

**Modificateurs** :
- -8 Dés (8 Folies)
- +3 Dés (Valeur Générosité)

**Total** : 5 + (2dD - 8dD + 3dD) = **5 - 3dD**

**Niv d'Épreuve** : +2 (négociation difficile)

## Résultat

**Jet** : `--0` = -2
**Total** : 5 - 2 = **3**
**3 ≥ 2 ?** Oui ! **Succès marginal.**

**Conséquence** : Le marchand accepte, mais à un prix élevé. Vous gagnez **1 Marque** dans [Négociation] (succès, pas d'échec donc pas de Souffrances).

## Si Vous Aviez Échoué

- **Échec** → 2 Rancœurs (liées à VOL)
- **10 Souffrances totales** → Rage Niv 1
- **Jet 1d6 > 1** pour agir rationnellement
- **Mais aussi** : +1 Marque dans [Négociation] (échec = expérience)

## La Beauté du Système

Même en échec, vous apprenez. Vos Souffrances créent des situations intéressantes. Vos Valeurs peuvent guérir vos Folies. Tout est interconnecté.
