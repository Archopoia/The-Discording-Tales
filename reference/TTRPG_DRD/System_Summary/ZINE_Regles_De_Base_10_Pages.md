# PAGE 1 : Le concept

## Des Récits Discordants

**Un jeu où l’échec forge le héros.**

Chaque échec vous rapproche de la maîtrise. Les Souffrances ne sont pas des punitions, mais des moteurs narratifs. À 10+ Souffrances, vous entrez en Rage ; à 15+, Évanouissement. Chaque Souffrance vous apprend aussi quelque chose.

**Les 3 principes**

1. **L’échec = progression** — Chaque échec dans une action possible donne une Marque d’expérience. 10 Marques = Réalisation = +1 Dé dans la Compétence.
2. **Souffrances = moteurs** — Les 8 Souffrances créent situations, choix et dilemmes. Comment gérez-vous votre Rage ? Vos Folies ? Vos Rancœurs ?
3. **Personnalité mécanique** — Vos Valeurs guérissent vos Folies. Vos Traits guérissent vos Rancœurs. La personnalité n’est pas cosmétique, elle est mécanique.

**Pour jouer**

- **1 Révélateur** (Maître de Jeu)
- **2–6 Éveilleurs** (Joueurs)
- **3 dés à 3 faces par joueur** (ou d6 : 1–2 = −, 3–4 = 0, 5–6 = +)

# PAGE 2 : L’univers

## Iäoduneï

Région de jeu : **Iäoduneï**, pays de dunes enneigées, eaux brûlantes et montagnes centrales (~200 000 habitants). Lieux clés : Dla’Adur, Lumioleï, Monts Ajolum, Déserts Soufflés, Grands Lacs Thermaux.

**Techno-traditionalisme** — Le monde mêle traditions et techniques. Les Motivations opposent l’ordre des **Gens** (société, coutumes) et des **Choses** (nature, technologie).

**Cosmogonie en bref** — La plupart des cultures croient vivre dans un univers concave (Cytocosmisme), sans création ex nihilo : une **continuité infinie**. Le monde invisible est expliqué par des **cordes tressées** ; les nœuds en sont les **Rils**. **Ô** est l’esprit rilique formant toutes les formes, éternellement entre douleur et soulagement. La **Rilie** (magie) naît de l’interaction avec les Rils, orchestrée par quatre forces (Tournoiement, Tension, Alignement, Torsion).

# PAGE 3 : Les dés

## Dés Discordants (dD)

**3 faces** : `+`, `−`, `0`.

**Résultat d’un jet** : nombre de `+` moins nombre de `−`.

**Exemples** : `++−0−` = 2 − 1 = **+1** ; `+++00` = **+3** ; `−−000` = **−2** ; `00000` = **0** (égalité).

**Sans dés spéciaux ?** d6 : 1–2 = `−`, 3–4 = `0`, 5–6 = `+`.

## Niveaux (Niv)

- **Niv d’Attribut** : de −5 à +5 (0 = moyen).
- **Niv d’Aptitude** : calculé à partir de 3 Attributs.
- **Niv de Compétence** : de 0 à 5 (Néophyte → Maître).
- **Niv d’Épreuve** : difficulté de l’action (−5 à +10+).

# PAGE 4 : Faire un jet

## Formule

**Résultat = Niv + Dés**

- **Niv** = Niv d’Aptitude + modificateurs (équipement, situation).
- **Dés** = 5 Dés de Chance + Dés de Compétence + Dés de Potentiel.
- **Succès** : Résultat ≥ Niv d’Épreuve.
- **Échec** : Résultat < Niv d’Épreuve → +1 Marque d’expérience.

## Exemple

Escalader une paroi (Niv d’Épreuve +2). Athlétisme Niv +1, [Grimpe] Niv 2 avec +3 Dés, équipement +1 Niv.

- Niv : +1 + 1 = **+2**
- Dés : 5 (Chance) + 3 ([Grimpe]) = **8dD**
- Si résultat ≥ +2 : succès. Si < +2 : échec, +1 Marque dans [Grimpe].

## Critiques

- **Échec critique** : tous les dés sont `−` → 5 Marques au lieu de 1.
- **Succès critique** : tous les dés sont `+` → effet spécial.

# PAGE 5 : Création rapide

## 1. Choisir un peuple

Aristois, Griscribes, Navillis, Méridiens, Hauts Ylfes, Ylfes pâles, Ylfes des lacs, Slaadéens, Tchalkchaïs (voir page 2).

## 2. Attributs de base

Chaque peuple a des Attributs de base. Ajoutez **+2, +1, 0, 0, 0, 0, −1, −2** où vous voulez.

**Les 8 Attributs** : **FOR** (Force), **AGI** (Agilité), **DEX** (Dextérité), **VIG** (Vigueur), **EMP** (Empathie), **PER** (Perception), **CRÉ** (Créativité), **VOL** (Volonté).

## 3. Aptitudes

Chaque Aptitude = moyenne de 3 Attributs :

- **Puissance** (FOR, AGI, DEX) — **Aisance** (AGI, DEX, VIG) — **Précision** (DEX, VIG, EMP) — **Athlétisme** (VIG, FOR, AGI)
- **Charisme** (EMP, PER, CRÉ) — **Détection** (PER, CRÉ, VOL) — **Réflexion** (CRÉ, VOL, FOR) — **Domination** (VOL, FOR, EMP)

# PAGE 6 : Compétences et Symbole

## Compétences

Choisissez **3 Compétences**. Donnez **+3 Dés** à l’une (familiale), **+2 Dés** à l’autre (raciale), **+1 Dé** à la dernière (populaire).

**Exemples** : [Armé], [Esquive], [Visée], [Grimpe], [Négociation], [Investigation], [Médecine], [Intimidation], etc. (72 au total, 3 par Aptitude).

## Symbole

Votre **Attribut le plus élevé** détermine votre Symbole (destinée / passion). Il donne **3 Compétences** avec **3 Marques Éternelles** chacune (elles comptent pour la progression).

Ex. FOR le plus élevé → Symbole Puissance → [Armé], [Désarmé], [Lutte].

## Valeurs et Traits

- **1 Moralité** : +6 Dés (héritée du peuple).
- **1 Motivation** : +3 Dés (envers Gens ou Choses).
- **1 Socialité** : +6 Dés (selon l’âge).

# PAGE 7 : Les Souffrances

## Les 8 Souffrances

Chaque Souffrance est liée à un Attribut :

1. **Blessures** (FOR) — 2. **Fatigues** (AGI) — 3. **Entraves** (DEX) — 4. **Disettes** (VIG)  
5. **Addictions** (EMP) — 6. **Maladies** (PER) — 7. **Folies** (CRÉ) — 8. **Rancœurs** (VOL)

## Comment les obtenir

**Échec dans une Épreuve possible** → Souffrances selon la gravité :

- Léger : 1–2 Dés de Souffrance (DS)
- Modéré : 3–5 DS → Séquelle passagère
- Sévère : 6–9 DS → Séquelle durable
- Critique : 10–11 DS → Séquelle permanente

Ex. tomber → Blessures ; nuit blanche → Fatigues ; faim → Disettes ; peur → Folies ; colère → Rancœurs.

## Dés de Souffrance (DS)

Les DS s’appliquent comme **dés négatifs** aux jets liés à l’Attribut affecté. Ex. 5 Blessures (FOR) → −5 Dés à tous les jets de Puissance.

# PAGE 8 : Seuils, Rage, Guérison

## Seuils

- **3–5 DS** : Séquelle passagère  
- **6–9 DS** : Séquelle durable  
- **10–11 DS** : Séquelle permanente  
- **15–20 DS** : Séquelle fatale

## Rage et Évanouissement

- **10+ Souffrances totales** : **Rage Niv 1** — Jet 1d6 > Niv de Rage pour agir rationnellement ; sinon, agir par instinct.
- **15+ Souffrances totales** : **Évanouissement / Démence** — Échanger 1 Niv de Rage à chaque nouvelle Souffrance ; Jet 1d6 > Évanouissement pour pouvoir agir.
- **21+ Souffrances** : **Vaincu** — Coma ou folie permanente.

## Guérison

- **Valeurs** : guérissent les Folies (CRÉ).
- **Traits** : guérissent les Rancœurs (VOL).
- **Traitements** : guérissent les autres Souffrances.
- **Résistance** : utiliser une Compétence résistante pour réduire les Souffrances reçues.

# PAGE 9 : Combat et Rilie

## Combat (simplifié)

- **Clins** : 1/3 de seconde. Combat en temps réel.
- **Clins d’Alerte** : déterminent qui peut Proagir (attaquer, déplacer, manipuler).
- **Réactions** (Bloquer, Parer, Esquiver) au **dernier Clin** de l’attaque ennemie.
- **Niv d’Ébranlement** : accumulé par les Réactions ; réduit vos jets (ex. 3 Ébranlement → −3 Niv).
- **Dégâts** : attaque réussie → Blessures selon la marge de succès (ex. +1 à +2 : 1–2 DS ; +3 à +4 : 3–5 DS ; +5+ : 6–9 DS).

## Rilie (magie)

La magie naît de l’interaction avec les **Rils** (formes fondamentales du monde), orchestrée par quatre Tétrarques : Tournoiement, Tension, Alignement, Torsion. Les **Honorations Riliques** sont des actions rituelles (sacrifices, serments, offrandes) pour obtenir des effets (guérison, résistance, dégâts, etc.). À révéler comme Compétence ; voir livre / `07_Rilie_Magie.md` pour les cercles et familles riliques.

# PAGE 10 : Expérience et exemple

## Expérience

- **1 Marque** par échec dans une Épreuve possible (5 si échec critique).
- **Marques Éternelles** (Symbole) : ne comptent pas pour la progression.
- **10 Marques** (hors Éternelles) dans une Compétence = **Compétence Éprouvée** → +1 Dé à la Compétence, Marques gratuites ailleurs, +1 à l’Attribut associé.
- **Distinction** = somme des Niv d’Attributs ; à chaque +10 Distinction : +1 à un Attribut et Découverte d’Attribut.

## Exemple de jeu

Négocier avec un marchand (Niv d’Épreuve +2). Charisme +3, [Négociation] Niv 2 avec +2 Dés, 8 Folies (−8 Dés), Valeur Générosité +3 Dés.

- Base : 5 + (2dD − 8dD + 3dD) = **5 − 3dD**. Jet `−−0` = −2 → Total **3**. 3 ≥ 2 → **Succès marginal.**  
Si échec : Rancœurs (VOL), possible Rage à 10+ Souffrances ; dans tous les cas, échec = +1 Marque dans [Négociation].
