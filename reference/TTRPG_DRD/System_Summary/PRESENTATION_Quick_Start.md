# Des Récits Discordants – Quick Start

*Sources : résumés 01–14 et LORE ; terminologie conforme au livre (Révélateur, Éveilleur, Niv, dD).*

## Le Concept en 30 Secondes

Dans **Des Récits Discordants**, chaque échec vous rapproche de la maîtrise. Vos Souffrances (Blessures, Fatigues, Folies, etc.) ne sont pas des punitions, mais des moteurs narratifs. À 10+ Souffrances, vous entrez en Rage. À 15+, Évanouissement. Mais chaque Souffrance vous apprend aussi quelque chose.

La magie (Rilie) naît des émotions d'une déesse éternellement torturée par 4 forces cosmiques. Vos Valeurs guérissent vos Folies. Vos Traits guérissent vos Rancœurs. Votre personnalité n'est pas cosmétique, elle est mécanique.

## Création Ultra-Rapide (5 minutes)

### 1. Choisir un Peuple
- **Aristois** : Guerriers souverains (FOR+, EMP+)
- **Griscribes** : Sages imbriqués (VOL+, CRÉ+)
- **Slaadéens** : Artistes audacieux (EMP+, DEX+)
- **Tchalkchaïs** : Altruistes sacrificiels (FOR+, EMP+, VOL+, VIG+)

### 2. Attributs de Base
Chaque Peuple a des Attributs de base. Ajoutez +2, +1, 0, 0, 0, 0, -1, -2 où vous voulez.

**Les 8 Attributs** : Force (FOR), Agilité (AGI), Dextérité (DEX), Vigueur (VIG), Empathie (EMP), Perception (PER), Créativité (CRÉ), Volonté (VOL)

### 3. Une Compétence à +3D
Choisissez une Compétence liée à votre Peuple et donnez-lui +3 Dés.

**Exemples** :
- Aristois : [Armé], [Chevauchement], [Puissance]
- Griscribes : [Investigation], [Ingénierie], [Médecine]
- Slaadéens : [Inspiration], [Débrouillardise], [Chant]

### 4. Un Symbole
Votre Attribut le plus élevé détermine votre Symbole (destinée/passion). Il donne 3 Compétences avec 3 Marques Éternelles chacune.

### 5. Une Valeur
Choisissez une Valeur liée à votre Peuple (Moralité) avec +6 Dés.

### 6. C'est Tout !
Vous avez un personnage jouable. Le reste se découvre en jouant.

## Les Dés (dD)

**Dés Discordants** : 3 faces : `+`, `-`, `0`

**Résultat** : Nombre de `+` moins nombre de `-`

**Exemple** : `++-0-` = 2 - 1 = **+1**

## Un Jet Simple

**Base** : Niv d'Aptitude + Niv de Compétence + Dés de Compétence

**Exemple** :
- Puissance Niv +2
- [Armé] Niv 1, +3 Dés
- Jet : 2 + 1 + 3dD = **3 + résultat des dés**

**Succès** : Résultat ≥ Niv d'Épreuve (difficulté)

## Les 8 Souffrances

Chaque échec dans une Épreuve Possible peut donner des Souffrances :

1. **Blessures** (FOR) : Dégâts physiques
2. **Fatigues** (AGI) : Épuisement
3. **Entraves** (DEX) : Empêchements
4. **Disettes** (VIG) : Faim, soif
5. **Addictions** (EMP) : Dépendances
6. **Maladies** (PER) : Affections
7. **Folies** (CRÉ) : Troubles mentaux
8. **Rancœurs** (VOL) : Ressentiments

**Dés de Souffrance (DS)** : Appliqués comme Dés Négatifs aux jets liés.

**Seuils** :
- **3-5 DS** : Séquelle Passagère
- **6-9 DS** : Séquelle Durable
- **10-11 DS** : Séquelle Permanente
- **15-20 DS** : Séquelle Fatale
- **10+ Souffrances totales** : Rage
- **15+ Souffrances totales** : Évanouissement/Démence
- **21+ Souffrances** : Vaincu (Coma/Folie)

## L'Expérience

**Marques** : 1 par Échec dans une Épreuve Possible (5 si Échec Critique)

**10 Marques** (moins Marques Éternelles) = Compétence Éprouvée

**Réalisation** : +1 Dé à la Compétence, Marques gratuites ailleurs, +1 à l'Attribut associé

## Combat (Simplifié)

**Clins** : 1/3 de seconde. Combat en temps réel.

**Clins d'Alerte** : Déterminent qui peut agir de manière proactive.

**Proactions** : Actions offensives (attaquer, déplacer)

**Réactions** : Bloquer, Parer, Esquiver (au dernier Clin de l'attaque)

**Niv d'Ébranlement** : Accumulé par les Réactions, réduit vos capacités.

## La Rilie (Magie)

La magie naît de l'interaction avec les **Rils** (formes fondamentales du monde), orchestrées par 4 forces :

- **iôHôi** : Tournoiement (Électromagnétisme)
- **sôIôs** : Tension (Gravité)
- **môSôm** : Alignement (Lumière)
- **hôMôh** : Torsion (Transmutation)

**Honoration Rilique** : Actions rituelles pour interagir avec les Rils.
